🔗 Connect Your Gmail

This CRM reads your Gmail inbox to surface accounts, past orders, and contact history. All processing happens in your browser — nothing is stored externally.

1
Go to console.cloud.google.com → Create a project → Enable Gmail API + People API
2
OAuth consent screen → External → add mpwsports@gmail.com as a test user
3
Credentials → OAuth 2.0 Client ID → Web Application → add this page's URL as an authorized redirect URI
4
Paste your Client ID below and click Connect
